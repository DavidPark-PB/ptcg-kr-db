# 개요

이 폴더에 있는 코드는 한국 포켓몬 카드 공식 홈페이지에 있는 제품정보탭을 스크래핑하는 것을 목표로 합니다.

https://pokemoncard.co.kr/card/category/info1

또한, 여러 제품이 한가지 제품으로 되어있는 경우를 나누는 것도 목표로 합니다.

# 1. scrape_product_info.py

이 파일은 각 제품의 페이지를 방문하여
필요한 정보를 가져오는 것을 목표로 합니다.

방문하는 페이지의 url은 다음과 같은 구조를 가집니다
> https://pokemoncard.co.kr/card/{id},
> id \in 1~647

각 페이지에서 가져오는 정보는 다음과 같습니다.
- id : 제품정보 url 에 있는 id입니다.
- name : 상품이름입니다.
- cover_url : 상품의 이미지 url입니다. 예를 들면 카드팩의 포장지등 입니다.
- price : 상품의 가격입니다.
- contents : 상품에 들어있는 내용물입니다.
- caution : 상품 페이지에 반드시 있는 주의문구 입니다.
- type : 확장팩, 구축덱, 특별제품, 서플라이를 구분합니다. 각각 pack,deck,special,supply이라 부릅니다
- releaseDate : 제품 발매일 입니다.
- url : 제품정보 url 입니다.

이렇게 제품정보를 얻으면 수많은 이상한 정보도 같이 딸려옵니다.
이는 포켓몬 코리아에서 이벤트 정보를 기재하는 페이지의 url과
제품정보를 기재하는 페이지의 url을 혼용하기 때문입니다.
이를 적절히 걸러주시면 됩니다.

# 2. split_product_info.py

이 파일은 카드제품(확장팩, 구축덱, 특별제품)만을 가져오고,
서플라이제품은 다른 파일로 분리합니다. 

그리고 여러 제품이 한가지 제품으로 되어있는 경우를 나눕니다.
숫자가 숫자다보니, scraping/prod_check/ 의 compare_prod_data.py를 이용해서
원래 나눠있어야 할 것이 붙어있는걸 찾았습니다.

예를 들면, 

https://pokemoncard.co.kr/card/124

에선 '울트라썬' 이라는 확장팩과 '울트라문' 이라는 확장팩을 하나의 제품정보로 묶었습니다.
이런걸 나눈겁니다.

또한, 이름의 공백등이 잘못 되어있는것도 수정합니다.

코드 자체는 상당히 원시적으로 하드코딩되어있습니다...
